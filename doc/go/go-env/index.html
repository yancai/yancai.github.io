  3. 配置环境变量
    Linux, Mac

    Windows
    新建环境变量GOHOME，值为Go安装位置
    新建环境变量GOPATH，设置并创建一个路径，例如：D:\AppData\go
    PATH中追加：;%GOHOME%\bin;%GOPATH%\bin;

  4. 验证
    命令行中输入go version可以正常查看go版本信息
